# Hayden Bohlen Portfolio

A focused one-page case study for Hayden Bohlen’s Claude Corps application, centered on BotMentor and its Nebraska Robotics Expo origin.

## Development

```bash
npm install --include=dev
npm run dev
```

## Quality gates

```bash
npm run typecheck
npm run build
```

## Deployment

Static Vite output is written to `dist/` and is suitable for Vercel. No backend, secrets, analytics, or runtime API are required.
